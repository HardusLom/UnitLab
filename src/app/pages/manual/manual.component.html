UnitLab User Manual

Everything you need to look up units, run conversions, explore SI prefixes and physics formulas, and test yourself with the quiz. Use the contents on the left to jump to any section.

Overview

UnitLab is a reference and conversion tool covering a wide range of physical quantities — length, mass, time, temperature, energy, and many more. It runs entirely in your browser, works offline once installed, and stores your favourites and history locally on your device.

The app is organised into six main pages:

  • Reference — a searchable, categorised table of every unit.
  • Converter — convert any value between two units of the same quantity.
  • Prefixes — scale a value across all 20 SI prefixes.
  • Formulas — a physics formula reference with variables and SI units.
  • Quiz — a multiple-choice quiz with three difficulty levels.
  • Saved — your favourite conversion pairs and recent history.

Getting started

No account or setup is required — open the app and you land on the Reference page. A typical first task is a quick conversion:

  1. Open the Converter from the menu.
  2. Pick a quantity (e.g. Length).
  3. Choose the unit to convert from and the unit to convert to.
  4. Type a value — the result updates instantly as you type.

Everything you mark as a favourite or convert is remembered automatically, so you can return to it later on the Saved page.

Navigation & search

The header is available on every page. It contains the app menu and a global unit search box.

  • Menu — switch between pages. On narrow screens, tap the ☰ button to open the menu.
  • Global search — start typing a unit, symbol, or quantity name. Results appear in a dropdown.

The search is context-aware about where it sends you:

  • On the Reference page, selecting a result filters the table.
  • On the Formulas page, it filters formulas by quantity.
  • Anywhere else, it opens the Converter pre-loaded with that unit.

You can drive the search dropdown entirely from the keyboard: ↑ ↓ to move, Enter to select, and Esc to clear.

Reference table

The Reference page lists every unit, grouped by quantity. Each row shows the unit name, symbol, the measurement system it belongs to, and how it relates to the base unit.

  • Search — narrow the table to matching quantities or units.
  • System filter — chips at the top show only SI, metric, imperial, US, or other units (or All Systems).
  • Category tabs — switch between categories of quantities, or pick All to see every category at once.
  • ⓘ info toggle — each category has an info button that expands a short description of what the quantity measures.
  • ⇄ button (and the unit name link) — jumps straight to the Converter with that unit loaded.
  • 𝐖 link — where available, opens the unit's Wikipedia article in a new tab.

Converter

The Converter performs live conversions between any two units of the same quantity. Results update as you type — there is no convert button to press.

Type in either the From or the To field — the other side recalculates instantly. Use the ⇄ button between the fields to swap the two units.

  • Compound units — some results also display as a compound value, e.g. 5 ft 9 in or 1 h 30 min.
  • Formula — shows the exact expression used for the conversion, e.g. °F = °C × 1.8 + 32.
  • Size comparison — a visual bar showing the relative size of the two units and their ratio.
  • All units table — your value expressed in every unit of the quantity; tap any row to make it the To unit.
  • Copy result / Copy URL — copy the result to your clipboard, or copy a shareable link to the exact conversion.
  • Save pair — the ☆/★ button saves the conversion pair to your favourites for quick reuse.

Deep links: the Converter's address includes the quantity, both units, and the value. Copy the link to share or bookmark an exact conversion.

SI prefixes

The Prefixes page scales a single value across all 20 SI prefixes, from yotta (10²⁴) down to yocto (10⁻²⁴). Enter a value, pick its prefix and unit, and see the equivalent at every other prefix at once — handy for understanding orders of magnitude.

Formulas

The Formulas page is a searchable reference of common physics formulas. Each entry lists the expression, a short description, the variables involved, and their SI units. Use the search box to filter by name or quantity.

Quiz

The Quiz tests your knowledge with randomised multiple-choice questions.

  • Difficulty — choose Easy, Medium, or Hard.
  • Category filter — focus on a single category, or test across all of them.
  • Score bar — tracks your current Score, number Answered, current Streak, and Best score.
  • Best scores — your highest score is saved separately for each difficulty level.
  • Reset score — clears the current run to start fresh.

Saved

The Saved page collects your favourite conversion pairs and your recent history (the last 25 conversions). Tap any entry to reopen it in the Converter. Everything here is stored locally in your browser.

Tips & shortcuts

  • Use the global search to jump anywhere quickly without opening the menu.
  • Copy a Converter link to bookmark an exact conversion for later.
  • Mark frequently used conversions as favourites so they're one tap away on the Saved page.
  • On the Reference page, combine the search box with the system filter to narrow long lists fast.
  • The app remembers your last conversions automatically — no need to save them manually to find them again.

Install as an app

UnitLab is a Progressive Web App (PWA), so you can install it like a native app and use it offline.

  • Desktop (Chrome / Edge): click the install icon in the address bar, or open the browser menu and choose Install UnitLab.
  • iOS (Safari): tap the Share button, then Add to Home Screen.
  • Android (Chrome): open the browser menu and choose Install app or Add to Home screen.

Once installed, the core pages work without an internet connection.

FAQ

Where is my data stored?

Favourites, history, and quiz scores are stored in your browser's local storage on this device only. Nothing is sent to a server.

Will I lose my data?

Clearing your browser data, or using a different browser or device, will start you with a fresh slate.

How accurate are the conversions?

Conversions use exact factors where they are defined. Note that the Julian year (365.25 days) is used for the year unit.

Which page should I start on?

For a quick conversion, use the Converter. To browse and learn, start on the Reference page.
